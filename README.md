# Карта путешествий

Интерактивная карта мира (стиль Google Maps) для отметки посещённых стран.
Одна HTML-страница на [Leaflet](https://leafletjs.com/) — без сборки и зависимостей: достаточно открыть `index.html` в браузере.

- Посещённые страны подсвечиваются светло-зелёным.
- Слева — панель со счётчиком и списком стран (флаг + название).
- Наведение на любую страну показывает её название.

## Как отметить посещённые страны

Открой `index.html` и отредактируй **только** блок `CONFIG` в начале файла —
массив `visited` с кодами стран (ISO 3166-1 alpha-2):

```js
const CONFIG = {
  title:   "Карта путешествий",
  visited: ["RU", "DE", "FR", "JP", "US", "TR", "TH"],
};
```

Добавь нужный код в массив, сохрани, обнови страницу. Коды — из таблицы ниже.
