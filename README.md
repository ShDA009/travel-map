# Карта путешествий

https://shda009.github.io/travel-map/

Интерактивная карта мира + регионы России на [Leaflet](https://leafletjs.com/) — без сборки, только открыть `index.html`.

## Как редактировать

Всё в одном файле — `config.js`:

```js
const MAP_CONFIG = {
  visited: ["RU", "DE", "TR"],   // посещённые страны — ISO 3166-1 alpha-2
  cities: [
    { name: "Москва", lat: 55.7558, lng: 37.6173, region: "Москва" },
  ],
  ...
};
```

- `visited` — коды стран из таблицы ниже
- `cities` — посещённые города; `region` должен точно совпадать с названием из GeoJSON (таблица ниже)
